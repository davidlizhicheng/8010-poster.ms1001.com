# poster.ms1001.com 上传清单

## 服务器上先删（或覆盖）

| 路径 | 原因 |
|------|------|
| `/www/wwwroot/poster.ms1001.com/.env` | 本地误传 `PORT=8010` 会导致 502 |
| `server-8010*.err.log` | 本地调试日志，无用 |

**不要删：** `data/poster.db`（已有用户数据时保留）

---

## 本地上传时排除（不要打包进 zip）

- `.env`（根目录，PORT=8010 本地专用）
- `.env.example`
- `data/poster.db`
- `data/config.json`（用 `deploy/config.production.json` 代替）
- `data/apiclient_key.pem`
- `data/payment-screenshots/*`（除 `.gitkeep`）
- `outputs/*`（除 `.gitkeep`）
- `__pycache__/`、`*.pyc`
- `server-8010*.err.log`
- `test_*.py`

---

## 必须上传

整个项目目录，重点文件：

- `server.py`、`poster_platform.py`、`wechat_pay_v3.py`
- `web/` 全部
- `deploy/` 全部（含 `production.env`、`config.production.json`）
- `requirements.txt`

---

## 上传后在服务器执行（复制粘贴）

```bash
cd /www/wwwroot/poster.ms1001.com

# 1. 用生产配置覆盖 .env（先 vi 改 ADMIN_PASSWORD 和 MINIMAX_API_KEY）
cp deploy/production.env .env
vi .env

# 2. 首次部署或 config 为空时
cp deploy/config.production.json data/config.json
vi data/config.json   # 填 api_key

# 3. 权限 + 安装 + 验证
chown -R www:www data outputs .env
chmod 640 .env
bash deploy/install.sh

curl -s http://127.0.0.1:8010/api/config | head -c 200
echo
```

返回 JSON 且含 `"auth_base_url":"https://ai.ms1001.com"` 即成功。
